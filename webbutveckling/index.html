Responsive Web Design (RWD)

Vad är Responsive Web Design? Betyder att layouten ändras baserat på den enhet (mobil, tablet, desktop) som använder sitter på. Samma HTML-element (i stor utsträckning). Brytpunkter definerar i CSS med hjälp av mediaqueries. Mobile-first Är en strategi där man skapar layout för mobilen först och sedan för desktop. Varför? Många hävdar att detta approach fokuserar mer på huvudinnehållet. Du behöver oftast mindra CSS-regler för mobil och det är sedan enklare att lägga till CSS-regler för desktop än tvärtom. När du skriver CSS för mobil-first använder du min-width i dina mediaqueries istället för max-width

CSS Flexbox Använd flexibiliteten med CSS Flexbox. Använd en madiaquery och byt mellan flex-direction: row och flex-direction: column. Du kan även använda flex-wrap: wrap för att låta element som inte får plats "falla" ner till nästa rad. Använd även flex: <number> för att göra att element förhåller sig mindre/större i samma flex-container. Du kan även använda order: <number> för att byta ordning på element.

RWD: Tips och strategier

Tips #1

Glöm inte viewport-taggen!

Ska stå innanför <head>-taggen. Se länk

Tips #2

Använd Mediaqueries!

Tips #3

Undvik fasta dimensioner!

Använd relativa enheter som %, vh, wh, em och rem samt min-width/min-height samt max-width/max-height.Se länk

Tips #4

Gör bilder responsiva!

Sätt bilderna till 100% bredd så att de kan stretcha och samtidigt behålla proportionerna. Se länk

Tips #5

Designa utifrån innehållet!

Tips #6

Använd moderna layouttekniker!

Som CSS Flexbox och CSS Grid (när vi kommer dit :-)
